夜的回响

发表于 2023-04-02 分类于 随想

夜晚，有星空，也有灯火，人们却在白炽灯下忘记了仰望星空的梦。星空——那来自万光年外遥远的光束终会散去，那时还会有人想要抬起头吗？如果他抬起了，还能看到什么吗？幸而，大爆炸给予宇宙光亮的同时也带给物种以思想。当他独自一人时，当他快乐、悲伤甚至是悲喜交集时，夜空，清冷的夜空，洁净的夜空，神秘的夜空，就是他们的冥想室。即使他们的肉体化为尘土，他们的思想依然在时间的浪潮中闪闪发亮，他们的声音在后来的无数时刻回响。

中国现代社会的意义是缺失的。科学与哲学这俩看似毫不相干的东西自古以来是携手并进的，而我国科学的发展的思想基础更多建立于最初的那个社会主义理想之上。苏联在上个世纪80年代就已经解体，我国社会主义的继续发展失去了一个模板。如果按照上个世纪阶级斗争的思想来看，究竟谁才是如今的敌人？我们闯出一条路，选择新时代中国特色社会主义，当年被美好的理想掩盖住的思维缺失又该如何弥补？作为中国未来的中流砥柱，我们这一代将致力于解决这一问题；作为人类的一份子，我们将为社会发展提供模板，为人类科学发展做出贡献。

夜响，一位理科生，一位OIer，一位二次元，一位哲学的初探者的写字台。

我和电影

发表于 2024-01-06 分类于 随想

我与电影

2023年就这样过去了。打开一年积攒下来的相册，我想要翻出些什么，却发觉一切都是那么平凡：我做了所有如今看来应做的事，犯过了所有如今看来必须经历的错误，取得了一些如今看来顺其自然的成就。这么说来，它们也不平凡，如此跌宕起伏，却是精彩的每一天。也许正是我的生活已然充满着一种自由的气息，才让这一切显得司空见惯，这未尝不是一个好的现象——思想不受生活琐碎的束缚得以放得更远。而在这“平凡”生活里仍然熠熠生辉的，便是《纪念册》这部我参与制作的微电影。

制作一部电影这个想法从萌生到今天实现已五年有余，这其中的历程也几经波折。我06年出生，成长阶段横跨了现今流行数字智能的发展时间。小时候父母的卡片照相机是我最喜爱的玩具之一，可惜只能在外出时才能接触到。当短视频自媒体兴起时，我恰巧熟练把玩着智能手机并拥有着一颗拿到新东西就想彻底钻研明白的心。所以说，拍摄这件事对我而言从来都不陌生，电影也一样。从幼儿园到现在，我经历过DVD机、视频网站充会员、存“种子”存母带等一系列正流行、已过时又“文艺复兴”的事件。但让这两件事情最终联系在一起的还得归功于五年前的那个下午。

我一直好奇于十一学校的神奇究竟在于哪些地方。是设备更先进，老师更厉害还是食宿更优秀？当学长学姐们带着我们一众学弟学妹们完成了我人生中第一次狂欢节的宣传片时，我才意识到，原来我也有机会去拍摄一些让更多人可以看到的“视频”。但我的能力还不够，我参加了学校的影视剪辑与摄影课程，后来也拥有了人生中第一台属于我自己的相机，得到了一台属于我自己的也能更好支持我制作的电脑。当然这期间也或主动或被动地接下了一些小的任务，还有更多的电影观看以及一些影视制作知识的摄入，暂且不提了。

时过境迁，中考过去，我已经高一了。我试验性地接下了学校感恩日宣传片的导演工作，并出色地完成了任务，尽管最后传播的范围不大，但已然是对我能力的认可。于是我踌躇满志地看向了影视基地，报名这个课程的人不少，各自都有自己的想法。一颗绿草无法抵抗大漠的荒凉。一边是无知自傲，一边是逍遥懒散，一个兢兢业业认真做了好几年事的人就这样被拉扯来拉扯去，消磨掉了壮志豪情。我意识到面对真正复杂的工作，团队配合比个人能力更重要。我想要去找一个好的团队，但眼下唯一一个充满活力的剧组却拥有着令我难以直视的作品；我想要自己组建一个心目中的团队，但学业压得尚未适应的我喘不过气来。理想停滞了，那是我高中以来最混乱的时候。我不清楚自己要做些什么，我朝思暮想的女孩也离开了我。

恐怕鱼和熊掌不能兼得，想要完成理想，似乎总得失去些什么。种种评判人的数据告诉我，想要维持一个相对稳定的生活学习状态，我已经无法失去任何东西了。我试图忘记理想，我把它从备忘录删除，但总有一个声音在我耳边回响：高中三年的目标，怎么能说删就删呢？我顶着不适把声音压回脑袋中那片记忆的垃圾场，上个好大学，以后有的是时间。

高一的最后几个月，偶然感觉各种事情都稍微缓和了些，大概踏上正路了。红窗汇的下午，我闲适地走进了电影院，出于某种惺惺相惜的情感，我始终会支持校内电影社团的成果。而在电影最后一幕结束后，我心中燃起了昔日的悸动，这分明就是我心目中学生电影的样子啊，这分明就是一个能够符合我要求的剧组啊！我第一次被他人的热情活力所折服，自己心中的热情也绽放起来。看到了他们需要我的能力，我要用我的技术助力剧组的大家一起实现各自心中那个完美的理想。

我在剧组负责后期，是个不大不小的人物。我认识了志趣相投的人，见识到了他们大胆美好的构想。我珍惜这次宝贵机会努力精进自己技术，也帮助着大家。但我与剧组的大家连接并不紧密，尽管后期到现场绝对不是什么必需项，我还是带着那一丝不愿离开舒适区的羸弱——即使理想已然看见，我依然慵懒地躺在床上，用游戏结束着节假日的一天又一天。最终不得不提的是，票房大卖，学生影院那看似很多的座位实现了接连的满员。

满地都是六便士，他却抬头看见了月亮。2024是新的一年，希望自己能够对理想再努把力，不要永远沉溺于享乐主义的魔爪之中，忙碌永远不是你放弃理想的借口。

从《美的历程》与《科学革命的结构》谈思想革命

从《美的历程》与《科学革命的结构》谈思想革命

最近我在阅读《美的历程》与《科学革命的结构》这两本美学史与科学史上的巨著，其中美学发展与科学革命的相似性引起了我的关注。两本书尚未读完，只能在此透过李泽厚与托马斯·库恩两位先生的思路，简要谈谈思想革命的方式。

（一）概念的冲突

“概念”这一词语出现在老师们对于《美的历程》给出的任务书当中，其用意大概是让我们关注各种美的概念在各个时期的出现与发展。在前几章的阅读中，我个人有个直观的感觉，就是美的发展就是新概念超越旧概念的过程。在旧美学概念盛行的时代，新概念往往是非主流的，甚至是遭受迫害的。但新概念的优越性最终会被更多人所接受，替代原有的旧概念。随着社会文明程度的发展，这种现象愈加明显。青铜器以其怪异狞厉的形象为古代野蛮氏族提供了恐吓与保护两种神圣的宗教作用，在后来又解除了这种宗教束缚，融入了更多现实生活和人间趣味的元素。魏晋时期对秦汉以来外在权威的否定精神，在其流行后促进了内在人格的觉醒和追求，最终也作用在了美学自觉性上的革新。

而在科学的发展中也可以得出相似的结论。正如哥白尼、伽利略等人对于“日心说”的维护，与教会所拥护的“地心说”相对抗，最终占据了主流却又被更加符合观测结果的新学说所代替。但“超越”并不意味着所有的概念都是一个接着一个在单一的时间线上交替发展的。正如科学对于一个问题的研究会在同一时刻出现无数种用以解释该问题的思维范式，但最终夺得上风的只可能是其中一个在当时被视为最优解的范式，而剩余的则会被视为所谓哲学的一种。这并不意味着没有得到接受的就被带进了历史的坟墓，许多年后他们仍然有可能被重新发掘出来进行阐释，正如古希腊伊比鸠鲁学派认为时间与空间是离散的在近百年才被物理学家所论说。而美学中与魏晋类似的人格觉醒在先前的时代也并非没有出现过。所以单纯用“超越”一次来概括发展的进程是不够的，不如说美的发展与科学的发展是在概念的冲突中产生的。

（二）范式的利用

《论语》整理

发表于 2023-08-16 分类于 读书笔记

论语整理

以「仁」为中心，梳理不同的表述；然后向上关联「礼」，向下关联其它德行，比如「孝」。

2023.8.16 已更新完仁、礼、孝

仁

  1. “仁”的基础

    “孝弟者也，其为仁之本与！”——《学而篇·二》

    孝顺父母、敬爱兄长

    曰：“仁者先难而后获，可谓仁矣。”——《雍也篇·二十二》

    需要一个付出、收获的过程

    ”夫仁者，己欲立而立人，己欲达而达人。能近取譬，可谓仁之方也已。“——《雍也篇·三十》

    不要好高骛远，从身边人、身边事做起，推己及人

  2. “仁”的反面例子

    子曰：“巧言令色，鲜矣仁！“——《学而篇·三》

    花言巧语、伪善的面貌

  3. ”仁“的作用

    子曰：”人而不仁，如礼何？人而不仁，如乐何？“——《八佾篇·三》

    曰：”礼后乎？“——《八佾篇·八》

    ”依于仁“——《述而篇·六》

    礼乐等万事的基础

  4. 对待“仁”的态度

    子曰：“里仁为美。择不处仁，焉得知？”——《里仁篇·一》

    应当选择有仁德相随的环境

    子曰：“不仁者不可以久处约，不可以长处乐。仁者安仁，知者利仁。”——《里仁篇·二》

    仁德虽然可能会使人身处贫困，但也会使人长久地处于快乐之中。正因如此，仁者安于实行仁德，智者懂得利用仁德获取长远而巨大的利益。

    “好仁者，无以尚之；恶不仁者，其为仁矣，不使不仁者加乎其身。有能一日用其力于仁矣乎？我未见力不足者。盖有之矣，我未之见也。”——《里仁篇·六》

    每个人都有足够的力量实行仁德，实行它并喜爱它是最好的，即使厌恶它仁德依然可以帮助人远离不仁的食物。

    “君子去仁，恶乎成名？君子无终食之间违仁，造次必于是，颠沛必于是。”——《里仁篇·五》

    子曰：“回也，其心三月不违仁，其余则日月至焉而已矣。”——《雍也篇·七》

    这句讲应该长久地与仁德相伴，以仁心用事

  5. “仁”的特征

    子曰：“知者乐水，仁者乐山。知者动，仁者静。知者乐，仁者寿。“——《雍也篇·二十三》

    “仁者不忧”——《子罕篇·二十九》

    仁者乐观，沉静而长寿

    子曰：”仁远乎哉？我欲仁，斯仁至矣。“——《述而篇·三十》

    仁存于我们身边

礼

  1. “礼”的概念

    子曰：”人而不仁，如礼何？人而不仁，如乐何？“——《八佾篇·三》

    曰：”礼后乎？“——《八佾篇·八》

    仁是礼的基础

    子曰：“大哉问！礼，与其奢也，宁俭；丧，与其易也，宁戚。”——《八佾篇·四》

    与其铺张浪费，宁可朴素俭约；与其仪文周到，宁可过度悲哀。

  2. “礼”的作用

    有子曰：“礼之用，和为贵。”——《学而篇·十二》

    遇事都能做得恰当

    “知其说者之于天下也，其如示诸斯乎！”——《八佾篇·十一》

    使得治理天下非常容易

    “恭近于礼，远耻辱也。”——《学而篇·十三》

    不致遭受侮辱

    “齐之以礼，有耻且格。”——《为政篇·三》

    使人民有廉耻之心而且人心归服

    ”八佾舞于庭，是可忍也，孰不可忍也？“——《八佾篇·一》

    子曰：”夷狄之有君，不如诸夏之亡也。”——《八佾篇·五》

    礼是万事的基础

    孟懿子问孝。子曰：“无违。”……“生，事之以礼；死，葬之以礼，祭之以礼。”——《为政篇·五》

    可作为孝的上位概念

    ”约之以礼，亦可以弗畔矣夫！“——《雍也篇·二十七》

    约束人避免离经叛道

    子曰：“恭而无礼则劳，慎而无礼则葸，勇而无礼则乱，直而无礼则绞。君子笃于亲，则民兴于仁，故旧不遗，则民不偷。”——《泰伯篇·二》

    端庄避免劳倦，谨慎避免懦弱，勇敢避免盲目，直率避免尖刻，人们之间情感深厚。

  3. “礼”的正面例子

    子入太庙，每事问。——《八佾篇·十五》

    “不能以礼让为国，如礼何？”——《里仁篇·十三》

    多询问、懂礼让恰是慎重对待礼的表现

    “动容貌，斯远暴慢矣；正颜色，斯近信矣；出辞气，斯远鄙倍矣。”——《泰伯篇·四》

    严肃容貌避免别人的粗暴懈怠，端正脸色容易使人相信，说话时考虑言辞声调避免粗鄙错误。

    子曰：“麻冕，礼也；今也纯，俭，吾从众。拜下，礼也；今拜乎上，泰也。虽违众，吾从下。”——《子罕篇·三》

    节省但不倨傲

    “暮春者，春服既成，冠者五六人，童子六七人，浴乎沂，风乎舞雩，咏而归。”——《先进篇·二十六》

    从礼的美好生活

    孔子于乡党，恂恂如也，似不能言者。其在宗庙朝廷，便便言，唯谨尔。——《乡党篇·一》

    孔子在本乡的地方上非常恭顺，好像不能说话的样子。他在宗庙里、朝廷上，有话便明白而流畅地说出，只是说得很少。

    朝，与下大夫言，侃侃如也；与上大夫言，訚訚如也。君在，踧踖如也，与与如也。——《乡党篇·二》

    上朝的时候，君主还没有到来，同下大夫说话，温和而快乐的样子；同上大夫说话，正直而恭敬的样子。君主已经来了，恭敬而心中不安的样子，行步安详的样子。

    见齐衰者，虽狎，必变。见冕者与瞽者，虽亵，必以貌。凶服者式之。式负版者。有盛馔，必变色而作。迅雷风烈必变。——《乡党篇·二十五》

    孔子看见穿齐衰孝服的人，就是极亲密的，也一定改变态度，表示同情。看见戴着礼帽和瞎了眼睛的人，即使常相见，也一定有礼貌。在车中遇着拿了送死人衣物的人，便把身体微微地向前一俯，手伏着车前的横木，表示同情。遇见背负国家图籍的人，也手伏车前横木。一有丰富的菜肴，一定神色变动，站立起来。

    “以吾从大夫之后，不可徒行也。”——《先进篇·八》

    无论何事，遵从礼法。

  4. ”礼“的反面例子

    ”八佾舞于庭，是可忍也，孰不可忍也？“——《八佾篇·一》

    三家者以《雍》彻。——《八佾篇·二》

    子曰：“禘自既灌而往者，吾不欲观之矣。”——《八佾篇·十》

    子曰：”觚不觚，觚哉！觚哉！“——《雍也篇·二十五》

    颜渊死，门人欲厚葬之——《先进篇·十一》

孝

  1. “孝”的概念

事物观测形态研究（未完成）

发表于 2023-04-04 分类于 随想
这是一篇关于生活中的奇妙现象的研究，算是打开了个人博客中”为什么“这一议题的大门😁

研究背景：我们先来做一个思想实验

让我们先在一个安静的环境坐下，闭上双眼，把注意力集中到脑海当中。

现在请你在脑海中浮现一个现实里与你关系很好、无话不谈的异性的身影，观察他/她一段时间；

再在脑海中浮现一个与你关系一般，甚至你只是知道个名字、见过一两面而已的异性的身影，观察他/她一段时间。

你所观察到的他们的人影有什么不同吗？

有没有一种感觉，与你在现实里熟识的那位的身体模型更加立体，而另一位则趋向于平面？

虽然可能存在主观因素，但似乎好多人身上都出现了这种迹象。

这种事物观测形态的区别是由什么影响的？

本文章就是为了哲学为主，物理为辅，并以其他种种角度探索这个问题的答案。无奈在下才疏学浅，只是翻翻资料略解一二罢了。

猜想

可能的猜想一（维度角度）：

我们作为三维生物，如果要达成对于相同维度的事物的认识需要有更高维度的因素加入，比如四维的时间，因此我们长时间接触而熟识的人物在我们的脑海里更加立体；而由于三维生物对于二维事物本身就多了一个Z轴的高维度因素，因此三维事物的二维形象很容易在我们脑海里产生。

可能的猜想二（量子角度）：

没有人能一直决定着自己的外貌性格，所以从某种概念上外貌性格一类的东西都是量子态的，无法确定的。因为外界因素导致一些量子被他人观测到了，这些量子构成了他人的认知。

可能的猜想三（认知角度）：

我们的认知可以表达为一些目前未知的物质的组合体，与他人的接触会使这些“认知”物质像化学反应一样堆积、反应，最终形成一些新的物质。堆积就类似于我们对他人不断递进的认识，而反应则更加倾向于我们思考中无意间对于他人性格的推演。

读书笔记《苏菲的世界》：伊甸园与魔术师的礼帽

发表于 2023-04-02 更新于 2023-04-03 分类于 读书笔记

第一篇读书笔记来自于最近开始读的一本哲学启蒙书《苏菲的世界》。尽管里面的主人公苏菲的年龄也就十来岁的样子，但对于高中生的认知水平和理解能力是非常适合阅读的，算是作者的一个很棒的设计（很可能说明每个学生在接触哲学之前，由于与社会并没有过深的接触，对于世界的认知是接近于零的，说这本书适合所有的学生也不为过）。

这本书我将读两遍，第一遍记录下自己最初的理解，第二遍对记录稍加修改。

以下内容为第一次阅读时所作，言辞略显稚嫩，请多包涵。

苏菲收到了一封奇怪的信……

伊甸园和魔术师的礼帽是该书的前两章，大致从哲学是什么以及哲学家是什么这两个角度，以苏菲对来信的思考展开叙述。

每一个人类出生的时候都对世界充满着好奇心，婴儿挥舞着双手双脚，指着新奇的事物发出“唔噜”的声音；但学生及成年人身上就不会有这样的迹象。

无论是学生的学习还是成年人的工作，大多都是重复性极强的，持续的重复让人们形成习惯。习惯是琐碎的，习惯了世界的运转，也就失去了对世界的好奇心。重复的事情总有做完的那一天，就像学生走向社会、成年人退休以及人类最终的死亡，幸运的人在那时可能会重拾好奇心，但对于个体来说，如海一般的时间已然逝去，无法挽回；对于社会来说，它的可持续发展性是存疑的。

哲学家就是一群充满好奇心的人，他们自己探索世界的同时，也不忘记警示他人习以为常的危害。人人都应该成为自己的“哲学家”，多向世界问“为什么”，并尽心去回答。
